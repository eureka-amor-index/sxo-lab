# ✦ SXO Lab

**SXO Lab** is a living repository of experimental interfaces, tools, and prototypes exploring how humans interact with search systems, language, and algorithms.

Created by **Eureka Amor**, this lab documents small experiments at the intersection of:

Path: https://sxolab.com.ar/

• Search Experience Optimization (SXO)  
• Semantic systems  
• Digital identity  
• Creative technology  
• Experimental interfaces

Rather than a traditional portfolio, SXO Lab is designed as a **living signal** — a place where ideas are tested, visualized, and released in public.

---

# 🔬 Current Experiments

## Query Pulse Scout
**Status:** Beta  

A search experiment designed to detect optimization opportunities, semantic patterns, and content momentum across queries.

Focus areas:
- search signals
- semantic structure
- optimization detection
- intent mapping




---

## Cadáver Exquisito
**Status:** Live  

A poetic interface inspired by the surrealist game "Exquisite Corpse".  
Each phrase triggers another, creating an evolving chain of language.

Focus areas:
- generative writing
- collaborative language
- creative systems




---

## Emotional Constellations
**Status:** Concept  

A visualization experiment mapping keywords into emotional and semantic constellations.

Focus areas:
- keyword emotion
- search intent clusters
- data visualization


Each experiment lives in its own directory and can be expanded independently.
