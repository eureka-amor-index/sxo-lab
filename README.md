# ✦ SXO Lab

**SXO Lab** is a living repository of experimental interfaces, tools, and prototypes exploring how humans interact with search systems, language, and algorithms.

Created by **Eureka Amor**, this lab documents small experiments at the intersection of:

• Search Experience Optimization (SXO)  
• Semantic systems  
• Digital identity  
• Creative technology  
• Experimental interfaces

Rather than a traditional portfolio, SXO Lab is designed as a **living signal** — a place where ideas are tested, visualized, and released in public.
